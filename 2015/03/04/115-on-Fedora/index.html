Fedora21安装115网盘

115网盘有官方的Linux客户端，但是只有deb包。因此无法直接安装在Fedora上面。用alien转换成rpm包之后还是无法正常安装。所以我们只能另辟蹊径手动安装了。

首先去官网下载deb安装包，然后进入这个包所在的目录，比如Download目录。

我们可以新建一个文件夹，用了保存解压deb包之后的安装文件。然后将deb包拷贝过去，比如。

mkdir t
cd t
cp ../115网盘的安装包.deb .

然后我们解压这个包。

ar vx 115的安装包.deb

你会看到几个文件，其中data.tar.gz是我们需要的安装包，然后我们安装它（做这一步前请登录root或者自行加sodu）：

tar -xzvf data.tar.gz -C /

到这里我们就安装完了，你可以试着运行115pan看看效果。

你很有可能会遇到这个问题

/lib/ld-linux.so.2: bad ELF interpreter: No such file or directory

解决办法是安装这个包，可以运行一下命令：

yum -y install glibc.i686
