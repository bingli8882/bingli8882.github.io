Ubuntu上安装配置Chrome Remote Desktop

Google官方的Chrome Remote Desktop是一个非常不错的远程桌面系统。他可以通过Google Account 链接任何地方的计算机，实现远程操作，并且支持Windows，Mac和Linux。特别是配合ChromeBook使用的话，大大增强了ChromeBook的扩展性。性能与便携性、电池寿命一直是笔记本电脑的主要矛盾。ChromeBook是未来笔记本电脑一个不错的发展方向，将主要运算放在云端，随身设备仅仅作为一个接入点。即满足了便携性又不用过于顾及性能，一举两得。可是就目前而言，很多情况下，用户仍然无法完全抛弃传统电脑，那么远程桌面，虚拟机之流就成为了一个不错的解决办法。本文讲简单介绍Chrome Remote Desktop在Ubuntu Linux上的安装方法。

在安装Remote Desktop之前需要安装官方的Chrome浏览器，这个很简单可以去Chrome的主页现在deb安装包。下载完成后双击安装就可以了。

然后登陆Chrome App Store安装Remote Desktop。安装完成后打开app，然后登陆google account之后，你会看到两个选项。

  • Remote Assistance
  • My Computer

##与他人共享电脑

Remote Assistance是让任意两台计算机相连。比如你去连接你朋友的电脑，或者你朋友连接你的电脑。首先由被连接的一方点击Share按钮，然后你会看到下图的弹出窗口。

中间的一串数字就是你计算机的临时ID，你将这串数字告诉需要连接你电脑的人，他们就就可以使用它来找到你的电脑。不必担心安全问题，这串数字是随机生成的，当你不需要连接的时候，关闭这个app，别人就找不到你的计算机了，而下次在共享你的计算机时，你会获得另外一串数字。

如果需要连接另一台计算机，你需要点击Access按钮，然后会弹出如下窗口。

输入对方的那串数字你就可以连接对方的电脑了。

##配置自己的电脑

如果你需要远程连接你自己家的电脑，这样很不方便。若是无人在被叫电脑前值守，你根本无法完成连接。因为就算你连接上了，每隔几分钟，Chrome就会自动询问是否还要继续让别人连接你的电脑，若是无人应答，连接窗口就会锁死。

所以如果你想在无人值守的情况下连接自己的电脑，你就需要第二种方法。那就是使用第二个选项My Computer。首先需要配置你需要被连接的电脑，然后就可以使用你自己的Google账号在任何地方都可以登陆你的计算机。

以Ubuntu为例，首先你需要下载安装Chrome Remote Desktop的本地安装包，64位的deb安装包,32位的deb安装包。

安装完成后，你还需要进行一些配置。

首先你需要在你的home文件夹里创建 .chrome-remote-desktop-session文件。

然后去这个文件夹/usr/share/xsessions/里面查看你desktop的启动命令。

以我的计算机为例。

cd /usr/share/xsessions/
ls

之后我会看到如下的结果。

pantheon.desktop

在这个文件夹里面只有pantheon.desktop文件(你的电脑可能会是其他的文件名)，用vim或者gedit之类的编辑器打开这个文件，找到一行以Exec开头的代码。我的是这样的。

Exec=gnome-session --session=pantheon

记住=之后的部分，然后打开你之前创建的.chrome-remote-desktop-session，然后输入下列代码。

exec /usr/sbin/lightdm-session "gnome-session --session=pantheon"

双引号之内的部分就上一个文件=之后的部分。

这时候在打开Chrome Remote Desktop，你就能看到 Enable remote connections.的选项。点击这个选项然后配置你的PIN码。之后你在其他电脑上打开Chrome Remote Desktop然后登陆你的Google账号就能看到你之前配置好的电脑显示在My Computer之列了。选择它输入PIN码就能实现远程连接了。

##调整分辨率

连接之后你可能会遇到分辨率看着很不爽的问题。Chrome Remote Desktop在连接自己的电脑时，是创建一个虚拟的session。虽然它仍然使用着你在电脑上的账号，但是却有自己的profile，是一个虚拟机。默认的分辨率是1600×1200，估计现在很少有人的显示器分辨率是这个数值，那么怎么更改分辨率呢？其实只需要在你profile里面加一行代码就可以了。

进入自己的home文件夹，打开.profile文件。在最后添加如下代码：

export CHROME_REMOTE_DESKTOP_DEFAULT_DESKTOP_SIZES=1366x768

我的的分辨率是1366*768，大家可以根据自己显示器的分辨率自行调配。完成配置之后，你需要重启service。

sudo /etc/init.d/chrome-remote-desktop stop
sudo /etc/init.d/chrome-remote-desktop start

或者重启电脑也可以。
