Windows 8 手动更新 Windows 10

虽然说现在Windows对于我来说意义已经不是很大了，我平时的工作都是在Linux和新买Mac上进行的，但是偶尔也会用Windows玩玩游戏，而且这个频率现在时越来越低。之前Windows之所以难以割舍，主要还是因为需要用到office这个神器，可是自从买了Mac，这个需求也没了。前天去BestBuy逛，看到新版的Windows 10好像挺漂亮的，而且Windows 8可以免费升级，所以就想给电脑升级一下就当重装系统了。可是Windows 10发布也有几天了，迟迟没有收到更新的通知，所以就只能自己动手丰衣足食了。

手动升级的方法其实很简单，只是有一些小的细节需要注意。主要步骤如下：

  1. 获取Windows 10安装文件

    安装文件的ISO可以从官网下载。选择下载Windows 10的ISO即可，安装时会根据你当前的系统自动升级成为home或者pro的版本。

  2. 准备安装

    有了ISO之后我们就可以把它刻录成DVD。有一点需要注意，我们是不可以用这个DVD引导安装的，因为我们是升级安装。虽然我听说有的版本的ISO是可以从DVD引导升级安装的，但是我下载的版本选择升级的时候，安装程序提示我要用Windows引导安装。因此我们只能重新以正常方式进入Windows然后双击DVD里面的安装程序开始升级。

    这里我们可以发现，因为是在Windows内安装的，所以并不需要刻录成DVD。完全可以直接解压ISO文件到某文件夹（推荐在C盘根目录或者C盘以外创建文件夹，C盘是指当前Windows所在的硬盘），然后直接打开安装程序安装。

  3. 安装Windows

    安装过程很简单一路按照提示进行即可。因为我是本着重装系统的目的安装的，所以我选择了不保留任何文件。这样的话Windows会把所有的系统文件夹全都移动到Windows.old这个文件夹里面。然后安装新的系统文件，这里我们先不要管它。先根据提示一路安装下去再说。有人会问既然不想保留之前的文件，为什么不格式化重新安装呢？原因是我们这时候并没有Windows 10的cdkey，安装完无法激活，所以只能通过升级安装的方法安装。

  4. 安装之后

    和Windows 8一样，输入Microsoft的账号登陆系统。现在大家应该可以看到Windows 10漂亮的桌面了。安装完之后我们需要检查驱动程序的安装情况。可是我花了十几分钟的时间也没有找的控制面板在何方。太蛋疼了。最后我的方法是：开始菜单->setting然后在setting里面搜索control panel，然后才找到。也许有更简单的方法，但是请自行开发。找到缺失驱动的硬件，然后手动安装驱动，这里就不细说了。

    那个Windows.old的文件，我们是没有办法直接删除的。需要右键点击C盘->属性->磁盘清理，然后清理系统文件，最后选择之前的Windows版本开始清理。
