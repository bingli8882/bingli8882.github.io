美观的代码打印工具a2ps

a2ps 是一款开源代码打印工具，安装和使用都很方便。

Mac下可以从brew安装。

brew install a2ps

Linux下也可以通过各自的代码管理工具安装。

打印命令很简单。

a2ps file_name

就可以直接调用默认的打印机开始打印。

如果想打印到文件可以用-o flag，比如：

a2ps -o out.ps file_name

一个比较常见的问题是，纸张的尺寸不对。我们可以通过--medium flag 来调整。常用的纸张有 A4 Letter。我家的打印机使用的Letter的纸，所以打印的时候可以用如下的命令。

a2ps --medium=Letter file_name
